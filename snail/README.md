<h3 align="left">
  <p>A snail climbs a tree "A" foot a day. The snail then slips down "B" feet each night. The height of the tree is "H" feet.</p>
  <p>Write a program counting the number of days it will take for the snail to reach the top of the tree.</p>
  <ul>
    <li>The program reads a, b, h line by line. The input values are guaranteed to be positive integers.</li>
    <li>If the snail cannot reach the top of the tree, the message should be displayed: Impossible.</li>
   </ul>
